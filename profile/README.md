# UNUSE

> インタラクティブコンテンツを開発するクリエイターチーム

---

## 🎮 概要
この組織は新しいゲーム体験・展示を目的として活動しています。  
主に以下の分野の開発を行っています。

- 組み込みシステム
- 画像処理
- インタラクティブゲーム開発
